 Python Data Science Project – Sales Analysis & Visualization

Description:
This project analyzes a sales dataset to generate business insights using Python, Pandas, and visualization libraries (Matplotlib & Seaborn). It includes feature engineering and advanced analysis for profit, revenue, and customer demographics.

Dataset:

Sample dataset with columns: Customer_Age, Gender, Product_Category, Revenue, Profit

Stored in data/processed_sales_data.pkl

Features Implemented:

Basic Analysis

Summary statistics (mean, min, max, std)

Count of product categories and gender distribution

Visualizations

Age distribution histogram

Gender distribution pie chart

Revenue vs Age bar chart

Profit Analysis

Most & least profitable product categories

Profit margin calculation (Profit / Revenue)

Profit vs Revenue scatter plot

Sub-Category Analysis

Revenue & Profit by Product Category and Gender

Horizontal stacked bar charts for sub-category insights

Advanced Features

User input filtering by Product Category (ALL or specific category)

Dynamic visualization of filtered data
